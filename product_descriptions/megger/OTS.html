Megger OTS Series

Portable Oil Test Set

Product Overview

The Megger OTS Series is a portable oil test set designed for dielectric breakdown testing of insulating oils used in transformers, switchgear, and cables. Available in 60kV, 80kV, and 100kV models to meet various testing requirements and standards.

Key Features

  • Multiple Voltages: 60kV, 80kV, and 100kV models
  • Standards Compliance: IEC, ASTM, and other standards
  • Automatic Testing: Programmable test sequences
  • Result Storage: 1000+ test results memory
  • Touch Screen: Easy operation and setup
  • PC Connectivity: USB and Bluetooth for data transfer
  • Portable Design: Lightweight and rugged

Specifications

Voltage Range Up to 60kV / 80kV / 100kV
Electrode Gap 2.5mm standard (configurable)
Voltage Rise Rate 0.5 to 5 kV/s
Test Standards IEC 60156, ASTM D877/D1816
Memory 1000+ results
Interface Touch screen, USB, Bluetooth

Applications

  • Transformer oil testing
  • Switchgear insulating oil
  • Cable oil analysis
  • Oil quality assessment
  • Preventive maintenance
