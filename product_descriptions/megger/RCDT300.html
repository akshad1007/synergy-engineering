Megger RCDT300 Series

Residual Current Device Tester

Product Overview

The Megger RCDT300 Series are dedicated Residual Current Device testers for testing RCDs, RCCBs, and RCBOs. They provide fast, reliable testing of trip times and trip currents to ensure electrical safety protection devices are functioning correctly.

Key Features

  • Full RCD Testing: Tests all RCD types (AC, A, B, F)
  • Trip Time Measurement: High accuracy timing
  • Multiple Test Currents: 10mA to 1000mA range
  • Ramp Testing: Measures actual trip current
  • 3-Wire Testing: Phase to phase testing capability
  • Large Display: Clear, easy-to-read results

Specifications

Test Currents 10, 30, 100, 300, 500, 1000 mA
RCD Types AC, A, B, F, S
Trip Time Range 0 to 999.9 ms
Ramp Test Yes, 0.5x to 2x IΔn
Safety Rating CAT III 300V
Memory Optional data logging

Applications

  • RCD/RCCB testing
  • Electrical installation testing
  • Periodic inspection and testing
  • Safety compliance verification
