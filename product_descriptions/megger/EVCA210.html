Megger EVCA210

Electric Vehicle Charging Analyser

Product Overview

The Megger EVCA210 is a dedicated Electric Vehicle Charging Analyser for testing and commissioning EV charging stations. It simulates electric vehicle behavior to verify charging point functionality without needing an actual vehicle present.

Key Features

  • EV Simulation: Simulates Type 1 and Type 2 vehicles
  • Control Pilot Signal: Tests CP signal integrity
  • Proximity Pilot: Verifies PP circuit function
  • Fault Simulation: Tests EVSE safety systems
  • Power Measurement: Voltage, current, and power analysis
  • Portable Design: Rugged and easy to carry

Specifications

Connector Types Type 1 and Type 2
Voltage Range 85V to 265V AC
Current Range Up to 32A per phase
CP/PP Testing Full signal analysis
Display Color TFT screen
Data Storage Internal memory + USB

Applications

  • EV charging station commissioning
  • Periodic verification testing
  • Fault finding and diagnostics
  • Installation verification
